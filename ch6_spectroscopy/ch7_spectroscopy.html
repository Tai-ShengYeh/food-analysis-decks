光與物質 · Beer-Lambert 基礎
光射進去，
什麼出來了？
透光率 Transmittance
T = P / P₀
吸光度 Absorbance
A = −log(T) = −log(P/P₀)
A = 0 → T = 1（100%，完全透過）
A = 1 → T = 0.1（10%）
A = 2 → T = 0.01（1%）
  • P₀：入射光強度（incident radiant power）
  • P：透過光強度（transmitted power）
  • A：無因次量，對數尺度
Beer-Lambert Law · 定量核心
A = ε · b · c
ε
莫耳吸收係數
單位：L/(mol·cm)
物質固有屬性，隨波長變化
b
光徑長度
單位：cm
通常標準比色管 = 1 cm
c
濃度
單位：mol/L (M)
欲求量，由 A 計算得出
由 A 求 c：
c = A / (ε × b)
定量分析的核心計算式
  • 前提：稀溶液（濃度 < 10 mM）
  • 前提：在 λmax 測量，靈敏度最大
  • 前提：單色光（narrowband light）
📌 FD&C 色素 ε 值：
Yellow 5：ε = 27,300 L/(mol·cm) at 427 nm
Red 40：ε = 25,900 L/(mol·cm) at 503 nm
Blue 1：ε = 130,000 L/(mol·cm) at 629 nm
→ Blue 1 靈敏度最高，極微量即可測到
互動練習 · Interactive Calculator
Beer-Lambert 計算器
A = 0.450, ε = 27300
c = 0.450/(27300×1)
= 1.648×10⁻⁵ mol/L
= 16.48 μmol/L
若 A > 0.8 怎麼辦？
稀釋樣品（例如10倍）後重測，結果乘回稀釋倍數
若 A < 0.2 怎麼辦？
增加濃縮倍數、使用更長光徑（5cm, 10cm比色管）
最佳測量範圍
A = 0.2 至 0.8
為什麼？
A > 0.8 → T < 16%
偵測器接收光極少，雜訊比例大
A < 0.2 → T > 63%
大部分光穿透，難以精確測量微小差異
✅ A = 0.2–0.8（T = 16–63%）
誤差最小，精確度最高
Deviations · Beer's Law 偏差
三種 Beer's Law 偏差
🧪
真實偏差
True Deviation
高濃度（>10 mM）時，分子間交互作用改變吸收特性
→ 非線性 A vs c 曲線
解法：稀釋至線性範圍
🔦
儀器偏差
Instrumental Deviation
多色光（polychromatic radiation）同時照射
Beer's Law 僅對單色光嚴格成立
解法：窄通帶設定（≤2 nm bandwidth）
⚗️
化學偏差
Chemical Deviation
溶液中化學平衡（解離/結合），改變吸光物種
pH 影響最常見（如 pH 指示劑）
解法：緩衝液控制 pH
💡 實務上：先建立校正曲線（calibration curve）確認線性範圍，再進行定量分析
Calibration Curve · 校正曲線
如何建立校正曲線
  • 配製 5–8 個已知濃度的標準液（涵蓋待測濃度範圍）
  • 在相同 λ 測量各標準液吸光度
  • 以 A（Y 軸）vs c（X 軸）作圖，擬合直線
  • R² ≥ 0.999 為品質標準
  • 樣品 A 代入方程式求 c
線性回歸方程式：
A = slope × c + intercept
→ c = (A − intercept) / slope
⚠️ 理想上截距應近似 0（Beer's Law 過原點）
Standard Addition · 標準添加法
基質效應的解法
何時使用？
  • 複雜食品基質（complex matrix）干擾測量
  • 基質成分影響 ε 值
  • 無法配製與樣品完全相同基質的標準液
操作流程
  • 取 多份等量樣品，分別添加不同量的標準品
  • 測量各份 A 值，對添加量作圖（得直線）
  • 直線外推至 X 軸負截距 → 即為原樣品濃度

Fluorescence Principle · 螢光原理
吸收 → 轉換 → 發射
  • 分子吸收激發光（λex = 350 nm）→ 升到激發態 S₂
  • 快速無輻射鬆弛（IC）→ 落至 S₁ 最低振動能態
  • S₁ → S₀ 以光子形式放出能量 → 螢光（λem = 450 nm）
  • λem > λex（損失部分能量為熱）→ Stokes 位移
螢光強度（低濃度近似）：
F ≈ 2.303 · φ · ε · b · c · P₀
F = k · P₀ · c
φ：量子效率 (0–1)　奎寧 φ ≈ 0.546
📌 90° 偵測：偵測器垂直於激發光方向， 避免接收穿透激發光，大幅降低背景干擾
Stokes Shift · 選擇性的關鍵
激發光 ≠ 發射光
Stokes 位移定義：
Δλ = λem − λex
奎寧：450 − 350 = 100 nm
  • 靈敏度高：偵測器不受激發光干擾
  • 選擇性好：可分別設定 λex 和 λem，選擇特定螢光物質
  • 線性範圍廣：低濃度下 F ∝ c
  • 侷限性：只適用於有自然螢光（fluorophore）的物質
✨ 常見食品螢光物質：
奎寧 (350/450 nm)、維生素 B₁ (365/435)、
維生素 B₂ 核黃素 (450/530)、葉綠素 (436/670)

Derivative Spectroscopy · 導數光譜法
一階導數法：消除背景干擾
  • 對螢光光譜取一階導數 dF/dλ
  • 緩慢變化的背景螢光（廣峰）在導數後趨近於零
  • 在過零點（zero-crossing）測量 → 選擇性更好
  • 可同時定量兩種重疊螢光物質
Pandey (1999) 奎寧導數法：
dF/dλ|₄₂₂ = 0.186c + 0.037
dF/dλ|₄₇₇ = −0.127c − 0.034
R² = 0.9988 / 0.9976（兩個過零點）
直接發射法（對照）：
F(450nm) = 19.598c + 4.354
R² = 0.9983

Ch7 核心要點
UV-Vis 吸光度法
  • A = εbc：吸光度與濃度成正比
  • A = 0.2–0.8：最佳線性範圍
  • λmax 處測量：靈敏度最大
  • 三種偏差：真實／儀器／化學

  • F = kP₀c：低濃度線性

  • 90° 偵測，Stokes 位移
  • 量子效率 φ 決定螢光強度
  • 高濃度：inner filter effect
  • 適用：奎寧、維生素、農藥殘留
