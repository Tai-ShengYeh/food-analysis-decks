# 茶葉中多重元素檢驗方法 — 互動教學 HTML 簡報

依據 **TFDAF0032.00《茶葉中多重元素檢驗方法》**（衛福部食藥署公告方法）製作的單檔互動教學簡報，
用於食品分析課程「食品鑑別／產地溯源」單元。

主題：以 **ICP-MS 量測 14 種微量元素指紋** + **機器學習統計判別**，研判烏龍茶為臺灣茶或境外茶。

## 開啟方式

直接用瀏覽器打開 `index.html` 即可（單一檔案、AI 圖已 base64 內嵌，可離線、可分享 URL）。

| 操作 | 鍵 |
|------|----|
| 下一頁／上一頁 | `→` `空白` ／ `←` |
| 全螢幕 | `F` |
| 點畫面左右側 | 翻頁 |

## 23 張投影片（SOIL 三段式）

- **引起動機**：封面 → 為什麼要驗產地 → 元素指紋核心命題
- **維持注意**：方法全貌 → 14 元素（可排序表）→ 取樣磨粉 → 微波消化 → 前處理排序（互動）
  → ICP-MS 原理／條件 → 含量公式 → 計算闖關（互動）→ 指紋雷達圖
  → **基礎研究 (Tsai 2021)** → **沒有單一元素能定案**
- **喚起行動**：**PCA 真實示範（重繪 Fig 4）** → **互動切換 T/TW/TF/A 看分群** → **特徵選擇 + 動手做 dataset**
  → 從數字到產地（決策樹）→ 五種統計模型 → 品保與限制 → 自我檢核 → 結語

互動元件：**可排序的 14 元素表**、**前處理流程排序遊戲**、**含量計算闖關**、**指紋雷達圖**、
**PCA 主成分得分散布圖**（重繪自 Tsai 2021 Fig 4）、**可切換元素子集 T/TW/TF/A 的即時 PCA 散布圖 + LDA 正確率讀數**、**自我檢核清單**。

機器學習段落以真實論文 **蔡承祥等 2021《以元素特徵區別台灣茶葉與國外茶葉之初步研究》台灣農業研究 70(4):231–242** 為依據，
搭配 [`dataset/`](dataset/) 的可執行教學資料集（見該資料夾 README）。

> 雷達圖標示「示意」、PCA 散布圖標示「重繪自 Tsai 2021 Fig 4」——區分示意與真實，不偽造官方數據。

## 重建（唯一 source of truth 是 Python 腳本，勿直接改 index.html）

```bash
# 1) （選用）重新生成 AI 主圖到 generated/
python gen_images.py            # 需環境變數 FAL_KEY；FLUX.1 Dev

# 2) 重建簡報（讀 generated/*.jpg → base64 內嵌 → 輸出 index.html）
python build_tea.py
```

## 架構

- `build_tea.py` — 投影片內容與版面（**唯一可編輯來源**）。
  重用 `../soil_deck_common.py` 引擎（導覽／小遊戲／表格／Chart.js），
  但**追加深色「茶綠＋金」皮膚**（同樣的 class 名稱，後定義的 `:root` 覆蓋淺色），
  另對 Chart.js 深色軸色做單行安全 patch；共用引擎本身不被修改。
- `gen_images.py` — 平行生成 4 張主圖（封面／元素指紋／ICP-MS／結語）。
- `generated/` — AI 原圖備份（PNG/JPG）。
- `index.html` — 產出物（約 0.29 MB，含內嵌圖）。

風格：深色 `#0e1512` 背景 + 茶綠 `#5fd07f` 主色 + 暖金 `#e7ad4d` 輔色。
技術圖（四分法、ICP-MS 流程、判別決策樹）為內嵌深色 SVG；主視覺為 base64 內嵌 AI 圖。
