# 油脂分析 Fat Analysis — SOIL HTML 教學投影片（Nielsen Ch17）

3 小時課程用的單一 HTML 互動投影片，淺色學術風，含 6 個嵌入式小遊戲。
這是 9 章系列的**第一章範本**。

## 檔案
- `build_fat_deck.py` — **唯一 source of truth**。所有內容改這裡，不要直接改 `index.html`。
- `index.html` — 產出檔（單一檔、可離線、Chart.js 走 CDN）。

## 重新產生
```bash
python build_fat_deck.py      # -> index.html（34 頁）
```

## 開啟
直接用瀏覽器開 `index.html` 即可，或本機起伺服器：
```bash
python -m http.server 8777 --directory ch17_fat_soil
# 瀏覽 http://localhost:8777
```

## 結構（SOIL 三段式，34 頁）
- 🟢 引起動機（P1–8）：封面 → 大問句 → 重要性 → 定義 → 分類 → 含量圖 → 核心命題 → 🎮分類配對
- 🔵 維持注意（P9–26）：方法全覽 → 前處理 → 溶劑 → 🎮溶劑測驗 → Goldfisch → Soxhlet → 🎮萃取分類 → 水解 → GC → 🎮流程排序 → Babcock/Gerber → 儀器法
- 🟠 喚起行動（P27–34）：可排序比較表 → 決策樹 → 🎮決策挑戰 → 計算範例 → 🎮計算闖關 → 重點 → 自評清單 → CTA

## 6 個嵌入式小遊戲（前端即時計分，不寫資料庫）
1. 脂質分類配對（點選配對）
2. 溶劑特性即時測驗（MCQ ×4）
3. 萃取法分類（連續/半連續/不連續）
4. GC 流程排序（▲▼ 排序 ×7 步）
5. 決策挑戰（給食物選方法 ×5）
6. 計算闖關（冰淇淋 Mojonnier，輸入對答）
